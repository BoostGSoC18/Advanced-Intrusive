Detailed Description

template<class O1, class O2, class O3>
class boost::intrusive::segment_tree_base_hook< O1, O2, O3 >

Derive a class from this hook in order to store objects of that class in a segment tree.

The hook admits the following options: tag<>, void_pointer<> and link_mode<>.

tag<> defines a tag to identify the node. The same tag value can be used in different classes, but if a class is derived from more than one segment_tree_base_hook, then each segment_tree_base_hook needs its unique tag.

link_mode<> will specify the linking mode of the hook (normal_link, auto_unlink or safe_link).

void_pointer<> is the pointer type that will be used internally in the hook and the container configured to use this hook.
