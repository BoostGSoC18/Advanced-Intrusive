Detailed Description

template<typename ValueTraits, class SizeType, bool ConstantTimeSize, typename HeaderHolder>
class boost::intrusive::segment_tree_impl< ValueTraits, SizeType, ConstantTimeSize, HeaderHolder >

The class template segment tree is an intrusive container which can efficiently perform range operations.

The template parameter T is the type to be managed by the container. The user can specify additional options and if no options are provided default options are used.

The container supports the following options: base_hook<>/member_hook<>/value_traits<>, constant_time_size<> and size_type<>.
