Detailed Description

template<typename NodeTraits>
class boost::intrusive::segment_tree_algorithms< NodeTraits >

segment_tree_algorithms provides basic algorithms.

segment_tree_algorithms is configured with a NodeTraits class, which encapsulates the information about the node to be manipulated. NodeTraits must support the following interface:

Typedefs:

node: The type of the node that forms the circular list

node_ptr: A pointer to a node

const_node_ptr: A pointer to a const node

Static functions:

static node_ptr get_left_child(const_node_ptr &target);

static void set_left_child(const node_ptr &target,const node_ptr &left_child);

static node_ptr get_right_child(const_node_ptr &target);

static void set_right_child(const node_ptr &target,const node_ptr &right_child);
