◆ ~suffix_automata_member_hook()

template<class O1 , class O2 , class O3 >
boost::intrusive::suffix_automata_member_hook< O1, O2, O3 >::~suffix_automata_member_hook ( )

Effects: If link_mode is normal_link, the destructor does nothing (ie. no code is generated). If link_mode is safe_link and the object is stored in an suffix_automata an assertion is raised. If link_mode is auto_unlink and is_linked() is true, the node is unlinked.

Throws: Nothing.
